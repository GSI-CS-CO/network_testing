XenaBay configurations:

1. Test with XenaManager v1.96

1.1. Start XenaManager
1.2. Load 'xenabay_gsi_use_case_4_layers.xmcfg' configuration
1.3. Reserve ports and load their configuration, eg., port_dm_broadcast_100m.xpc
1.4. Optional: frame size can be edited in 'Stream Configuration Grid'
1.5. Traffic generation is done mainly in 'Global Statistics' tab.
1.5.1. Set stop time in 'Stop At:', eg., 5 min
1.5.2. Start the traffic generation by clicking on 'Start Traffic'
1.5.3. Save the measurement results by clciking 'Save' after the streaming is completed.
